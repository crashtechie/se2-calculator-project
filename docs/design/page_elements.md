### HTML Tags to identify for pulling data
    - Item Name: `<div class="druid-title">Steel Plate</div>`
    - Item Mass: `<div class="druid-data druid-data-mass druid-data-nonempty">35</div>`
    - Crafting Time: `<div class="druid-data druid-data-BaseTime druid-data-nonempty">0.2s</div>`
    - Fabricator Type: `<div class="druid-data druid-data-Location druid-data-nonempty"> ... Smelter 2.5 m ... </div>`
    - Materials and Quantities:
        ```
        <div class="druid-row druid-row-Item0" data-druid-section-row="Inputs">
            <div class="druid-label druid-label-Item0">
                <span>
                    <span style="white-space: nowrap;">
                        <a href="/wiki/Iron" title="Iron">
                            <img alt="Icon Item Iron.png" src="/images/thumb/Icon_Item_Iron.png/21px-Icon_Item_Iron.png?3d27a6" decoding="async" loading="lazy" width="21" height="21" data-file-width="128" data-file-height="128">
                        </a>
                    </span>&nbsp;
                    <a href="/wiki/Iron" title="Iron">
                        <span title="Iron">Iron</span>
                    </a>
                </span>
            </div>
            <div class="druid-data druid-data-Item0 druid-data-nonempty">35</div>
        </div>
        ```
    - Material Name: `<a href="/wiki/Iron" title="Iron">`
    - Material Quantity: `<div class="druid-data druid-data-Item0 druid-data-nonempty">35</div>`
    - Repeat for Item1, Item2, etc. as needed
    - Note: Some components may not have any materials listed.
    - Fabricator Type can be mapped as follows:
        - Smelter 2.5 m  -> Smelter
        - Assembler 5 m -> Assembler
        - Fabricator 10 m -> Advanced Assembler
        - Refinery 7.5 m -> Refinery
    - Crafting Time is given in seconds (s) and should be converted to a float value.
